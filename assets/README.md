# Assets

## icon.png

Notification icon for desktop notifications.

**Requirements:**

- Format: PNG
- Size: 128x128 pixels (recommended)
- Used by: `notify-send` (Linux), `terminal-notifier` (macOS)

**Note:** macOS `osascript` does not support custom icons. To use icons on macOS, install `terminal-notifier`:

```bash
brew install terminal-notifier
```
