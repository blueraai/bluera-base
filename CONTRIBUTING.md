# Contributing

Contributions to Bluera Base are welcome!

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/my-feature`)
3. **Make** your changes
4. **Test** using dogfooding: `claude --plugin-dir .`
5. **Submit** a pull request

## Development Setup

See [docs/development.md](docs/development.md) for setup instructions and project structure.

## Guidelines

- Follow the CLAUDE.md conventions in this repository
- Test commands and hooks before submitting
- Update documentation if adding new features
- Use conventional commit messages (`feat:`, `fix:`, `docs:`, etc.)

## Reporting Issues

- Use [GitHub Issues](https://github.com/blueraai/bluera-base/issues)
- Include Claude Code version and OS
- Provide steps to reproduce
